Chapter 3: Tools

Chapter 3: Tools

Tools Used:

  1. Git: A distributed version control system commonly used for tracking changes in source code during software development.
  1. PhpStorm: An integrated development environment (IDE) for PHP developed by JetBrains.
  1. Visual Studio Code (VS Code): A free source-code editor made by Microsoft for Windows, Linux, and macOS. It includes support for debugging, embedded Git control, syntax highlighting, and more.
  1. Laravel: A free, open-source PHP web framework intended for the development of web applications.
  1. Node.js: A JavaScript runtime built on Chrome's V8 JavaScript engine. It's used for building scalable network applications.
  1. WebSocket: A communication protocol that provides full-duplex communication channels over a single TCP connection. Used for real-time web applications.
  1. Apache2 Web Server: Open-source and free web server software that provides a secure and efficient HTTP server in line with current HTTP standards.
  1. MySQL: An open-source relational database management system.
  1. SQLite3: A C-language library that implements a small, fast, self-contained, high-reliability, full-featured, SQL database engine.
  1. Linux: A family of open-source Unix-like operating systems based on the Linux kernel.
  1. GitHub: A platform that provides hosting for software development and version control using Git.
